# 🏥 Family Health Tracker (C Project)

A comprehensive console-based health management system built in C as a first-semester project. This application helps track and analyze physical and mental health data for multiple family members.

---

## 🚀 Features

- 👤 Add, Update, Delete, and Search family members
- ⚖️ BMI Calculation with health category
- ❤️ Blood Pressure & Heart Rate Tracking
- 🩸 Blood Sugar & Cholesterol Analysis
- 🧠 Mental Health Tests:
  - GAD-7 (Anxiety Test)
  - PHQ-9 (Depression Test)
- 😴 Sleep Tracking:
  - Hours + Sleep Cycles
  - Age-based evaluation and suggestions
- 💾 Save records to a text file
- 📊 Export data to CSV format
- 🔍 Case-insensitive search functionality
- 🧹 Input validation for safe user input

---

## 🛠️ Tech Stack

- Language: **C**
- Concepts Used:
  - Structures
  - Dynamic Memory Allocation (`malloc`, `realloc`)
  - File Handling
  - String Manipulation
  - Modular Functions
  - Input Validation
